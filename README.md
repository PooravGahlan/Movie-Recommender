# 🎬 CineSense

A movie recommender that actually looks good. Pure Python, no HTML/CSS/JS.

---

## What's inside

**13 pages:**

| Page | What it does |
|---|---|
| Trending & Latest | Hot movies this week + newest releases |
| By Genre | Action, Comedy, Horror — pick your vibe |
| Mood Match | Tell it how you feel, get movie picks |
| Search | Find movies, actors, or directors |
| Surprise Me | Can't decide? Let the app pick |
| Top Rated | Best movies ever, filter by year & rating |
| Compare | Two movies side by side — stats, budget, revenue |
| AI Recommends | Personalized picks based on your watchlist |
| Trivia Quiz | Guess the release year |
| Watch Providers | Where to stream (Netflix, Prime, etc.) |
| Stats Dashboard | Watchlist analytics with charts |
| Poster Collage | Generate a poster grid, download as PNG |
| My Watchlist | Save movies, export CSV/JSON |
